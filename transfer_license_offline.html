Transfer a License from an Offline System

Transferring a license out of an offline system is a 2 step process.  First the transfer is started on the offline system and then completed on a system with a connection to the internet.

 

Step 1:

1.On the disconnected and licensed system, run the application.  

2.Select License from application menu.  

3.Click Release License to Transfer to another Machine button from the license dialog.

4.Click Yes when prompted.  
 
If the system is disconnected from the internet and the MECSOFT_OFFLINE environment variable is set (as described in Step 1 of License Activation on Offline Systems), a text file (*.txt) with a verification string will be created in the temp folder of this machine and will display on the screen.

If the text file (*.txt) does not display automatically, you can locate it under C:\Users\<username>\AppData\Local\Temp (example: <productname>OffLineTransfer.txt).  For easy access to this folder type %temp% on the address bar in the Windows File Explorer program.

5.Copy this text file (*.txt) to a portable drive or to a drive that is accessible from the internet-connected system.  This data file must be transmitted to the system that is connected to the internet to release the fulfillment for this activation.

6.Close the application.  
Step 2:

1.On the internet-connected system, run the MecSoftDisconnectedLicenseManager.exe.  

2.Select the Transfer License option on the left panel.    

3.Use the browse button to locate the text file (*.txt) that was created in Step 1 above and then select the Transfer License button.

4.This will contact the MecSoft's license server on the web and release the license for transfer.  You will see a confirmation message once the license transfer is successful.
