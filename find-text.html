The Find dialog

Use this dialog to Find text within your gcode file.  

Find What

Enter the text you wish to find and then pick Find Next to locate it.

Find Next

Go to the next location of the text.

Options

•Match whole word only:  Only complete words will be located.

•Match case: Use case sensitivity:  Only words that match the upper/lower case will be located.

Direction

•Search Up:  Search upwards in the g-code file.

•Search Down:  Search downwards in the g-code file.
Procedure

1.Select Find from the Edit tab or right-click and select Find from the menu.  The Find dialog will display.

2.Enter the text to find and then pick Find Next.
