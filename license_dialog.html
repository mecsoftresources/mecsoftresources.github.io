License Dialog

The License dialog allows you to manage your RhinoCAM software license.  

If you have purchased a multi-seat Floating License, DO NOT enter your network license Activate Code in this License dialog!  A Floating License must be activated from within the MecSoft Floating License Manager!

Select from one of the following topics:

•Running in Demo Mode

•Activate your Product  

•Check your License Status  

•Release your License  

•Re-register with another Activation Code
