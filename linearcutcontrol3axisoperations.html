Linear Cut Control, 3 Axis Operations

The Linear controls are only enabled when Cut Pattern (see above) is set to Linear.  Select from the following:

Start at Top / Start at Bottom  
Allows you to specify the start point for the Mixed cut pattern.

Angle of Cuts  
Allows you to specify the Angle of the Mixed cut pattern.

Cleanup Pass  
Check this option to cause automatic detection of all the corners that the tool could not reach between each pass. It will then add a toolpath based on the uncut area detected; either a linear cut in case of smaller areas or a cut that travels along the shape of the uncut area, when the area is large. This is applicable for Linear cut patterns.
