How to: Reset VisualCAD/CAM Settings?

To reset your application you would need to edit the windows registry,

Here are the basic steps:

1.Login as Admin.

2.Close all sessions of VisualCAD/CAM.

3.In the windows task bar search field type "regedit" (no quote marks).

4.This will display the registry editor.

5.On the left expand the folder named HKEY_CURRENT_USER

6.Then expand the sub folder named SOFTWARE.

7.Then expand the sub folder named MecSoft Corporation.

8.Then locate the sub folder named VisualCAD 2021, right-click on it and pick delete (this will reset VisualCAD).

9.Then if needed, locate the folder named VisualCAM 2021, right-click on it and pick delete (this will reset VisualCAM).

10.Close the registry editor.

11.Run VisualCAD/CAM.
