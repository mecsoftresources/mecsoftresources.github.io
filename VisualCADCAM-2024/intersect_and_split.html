Intersect & Split Mesh

This command intersects and splits two or more intersecting meshes. The first set of meshes may be pre-selected.  The first set of meshes will be intersected and split creating a third set of meshes.  The second set of cutting meshes will not be modified.

 

Your Meshes Must be Closed:  If any selected mesh does not form closed volume (i.e., there are any open edges) the command will fail.  This can sometimes occur with imported data files.  If this occurs, use the Auto Fix command on each mesh (independently) before using this command.  You can also use the Diagnose command to check for mesh errors.
