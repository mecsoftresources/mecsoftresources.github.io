Create a Solid Box

Create a Mesh Solid in the shape of a rectangular box.  See What is a Solid? for information about Meshes and Solids in VisualCAM.

Notes

1.The bottom of the box will lie in a base plane (parallel to the Construction Plane of the active view) with its corner at the first point defined.

2.If the second point does not lie on the base plane, it will be projected onto that base plane.
