Select Features for Machining Operations

Machining Features are regions defined under the Features tab in Machining Objects Browser. Creating machining features allows you to create toolpath operations from the those features to assist in automating your CAM work-flow.  For example, if your part has several pocket features, the system can use the feature information in your part to help automate the creation of pocketing toolpath operations.

In the example below, 1 Hole Feature (with 4 instances) and 1 Planar Face Feature are added to the Features tree.

Machining Features created under the Features tab in Machining Objects Browser

A Machining Feature MUST FIRST be created BEFORE a toolpath operation can be created from them.  Refer to the Features Tab of the Machining Objects Browser for creating Features.

After a toolpath operation is derived from a Machining Feature you can Select Features to add to the operation from the Control Geometry tab.

To create toolpath operations from Machining Features, refer to the Features Tab of the Machining Objects Browser.  Once an operation is created, pick the Select Features button to modify your selection.

To Select Features, when creating a machining operation, click Select Features under Control Geometry tab

The dialog will minimize while you select one or more features to machine.  Accepting the selection and the dialog will reappear with the features listed.

Selecting a region will include all the curves selected as regions.
