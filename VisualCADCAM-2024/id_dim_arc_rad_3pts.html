Create a 3 Point Arc Radius Dimension

Create a dimension label for an arc.  It measures the radius of the circle of which the arc is a part.  The letter ‘R’ appears in the label to show that it is measuring the radius.

An arrow specifies which arc is labeled.  The text is always horizontal.  Both are always in the plane of the arc. Both can be slid around the arc.  The units are included. Note: You can use points as input to this command.
