Circle, 3 Pts

Create a circle by picking three points on the circumference of the circle.

 

Command Prompts
Basic Procedure

 

Screen Pick

Command Input

Step 1

(Optional) Enter X, Y, [Z] coordinates for the first point.

Step 2

(Optional) Enter X, Y, [Z] coordinates for the second point.

Step 3

(Optional) Enter X, Y, [Z] coordinates for the third point.

Final


Notes

1.The circle will be created in a plane that passes through all three points.  These points may not be collinear.

2.The start of the circle will be at the first circumference point specified.
