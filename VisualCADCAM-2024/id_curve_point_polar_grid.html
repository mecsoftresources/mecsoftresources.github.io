Point, Polar Grid

Create points in a circular grid array.

 

Command Prompts
Basic Procedure

 

Screen Pick

Command Input

Step 1

 

 

 

 

Enter the number of points in the circular array.  The default value is shown in the Prompt.

 

Step 2

 

 

Enter the angle that a radial line through the first circumference point makes with the X axis.  The Default value is shown in the Prompt.

 

 

Step 3

(Optional) Enter X, Y, [Z] coordinates for the first point.

 

Step 4

(Optional) Enter X, Y, [Z] coordinates for the second point (determines the radius).

 

or...

(Optional) Enter the radius value directly.
