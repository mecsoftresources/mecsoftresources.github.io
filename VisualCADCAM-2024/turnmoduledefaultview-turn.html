TURN Module Default View

TURN Module uses the Top view as the default view. The coordinate system of this top view is setup to be aligned with the turning coordinate system. That is the origin of the screen is located at the center of the screen and the Z axis goes from left to right and the X axis goes from bottom to top. This display setup is not typical in design systems where the Top view is aligned with the XY axes of the world coordinate system.

This view setup is used in TURN Module to allow the turning center programmer to work in turning center coordinates rather than in the XY coordinates of the design system. The world coordinate system in VisualCAD is in XY coordinates.  All part modeling would still be done in XY coordinates. Loading the turn browser sets up the turn coordinate system.

It should be noted that this convention might sometimes be disorienting for users who are used to visualizing their design parts in the normal XY aligned display rather than the ZX aligned display.

TURN Module’s Top view is by default aligned with the ZX turning center coordinate system.
