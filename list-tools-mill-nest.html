List Tools

This allows you to List and Print your tools.

1.From the Tools tab of the Machining Objects Browser, select the List Tools button .

Note: The actual icons you see in this dialog will depend on what module and what configuration you are currently licensed to operate

From the Tools tab of the Machining Objects Browser, select the List Tools button

2.The button brings up all the tool properties associated with the tools currently recorded in the current session.  From the Cutting Tools Information dialog box, you can view or Print your Tool List.

From the Cutting Tools Information dialog box, you can view or Print your Tool List

3.Pick OK to close the dialog.
