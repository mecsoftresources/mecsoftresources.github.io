Entry/Exit tab, 5 Axis

The following Entry/Exit tab determines the way in which tool enters and leaves the part geometry. You specify Entry Motions, Exit Motions and Cut Connection parameters.  Refer to each of the sections below for more information.

 

Dialog Box: Entry/Exit tab

Dialog Box: Entry/Exit tab, 5 Axis Milling Operations
Entry Motions / Exit Motions

Type

Entry and Exit Motions can be defined as Tangential Arc, Vertical Tangential Arc, Horizontal Tangential Arc, Orthogonal Arc, Tangential Line and Orthogonal Line.

Tangential Arc

Vertical Tangential Arc

Horizontal Tangential Arc

Orthogonal Arc

Tangential Line

Orthogonal Line

Sweep Angle / Radius  

Selecting an arc entry motion allows you to specify the Sweep Angle and Radius.

Length

Setting entry motion to line requires you to specify the Length.

Ramp Height

Specifying a Ramp Height for entry and exit motions adds the specified height to the ramp motion.

Ramp Height = 0

Ramp Height = 0.25

Tool Axis Transitions  

This can be set to Fixed, Tangential or Tilted.

Cut Connections

Entry and exit motions can be applied to each Cut Connection.

Apply Entry at cut connections  

This applies entry motions for all cut connections.

Apply Exit at cut connections  

This applies exit motions for all cut connections.
