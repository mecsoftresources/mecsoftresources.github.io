License Activation on Offline Systems

Sometimes it is desirable to activate a license on a system with no access to the internet. MecSoft allows you to do this; it is a 3-step process, with the first step performed on the disconnected system, then the next on an internet-connected system. Finally, the resulting license must be transferred back to the disconnected system.

 

Step 1:

1.Set an environment variable MECSOFT_OFFLINE on the computer with no internet access:

a)Go to Start->Control Panel->System & Security ->System -> Advanced System Settings.  

b)From the System Properties dialog box that pops up click on the Environment Variables button at the bottom.  

c)In the Environment Variables dialog box that pops up, click on the New button under System variables.  

d)In the New System Variable dialog box that pops up, define
Variable Name = MECSOFT_OFFLINE
Variable Value = 1  

e)Pick OK to close Environment Variables dialog.  

f)Pick OK to close the System Properties dialog.

g)Close the Windows Control Panel.  

2.Run the MecSoft application and click Activate Paid Product.

3.Enter the Activation Code and click Register.  

4.Click Yes when prompted.

If the system is disconnected from the internet and the above environment variable is set, then the system will create a text file with the extension *.txt (example: <productname>OfflineRehost.txt) in the temp folder of the machine.

5.The text file would display on your screen. Copy this file to a portable drive or to a drive that is accessible from the internet-connected system.  The data in this file must be transmitted to the system that is connected to the internet to fulfill the activation code and to create license data.

If the text file (*.txt) does not display automatically, you can locate it under C:\Users\<username>\AppData\Local\Temp (example: <productname>OfflineRehost.txt).  For easy access to this folder type %temp% on the address bar in the Windows File Explorer program.

6.Close the application.
Step 2:

1.On the internet-connected system, run the MecSoftDisconnectedLicenseManager.exe.  

2.Use the browse button to locate the file that was created in Step 1 above.

3.Select Create License File option on the left panel.

4.This will contact the MecSoft's license server on the web and create a license file with extension (*.lic). The license file would display on your screen. Copy this file to a portable drive or to a drive that is accessible from the system not connected to internet.

If the license file (*.lic) does not display automatically, you can locate it under C:\Users\<username>\AppData\Local\Temp (example: visualcam.lic).  For easy access to this folder type %temp% on the address bar in the Windows File Explorer program.
Step 3:

1.Go to the computer that is not connected to internet where you had the environment variable set as described in Step 1 above.

2.Copy the license file (*.lic) created in Step 2 above to C:\ProgramData\MecSoft Corporation
By default the ProgramData folders in Windows hidden. Click here to learn how to show hidden files and folder.

3.Run the MecSoft application and from the application menu select License and this would display the license status.
