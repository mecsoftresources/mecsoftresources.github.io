Reverse Boring, 4 Axis

This method is used to reverse-bore a stepped hole that cannot be accessed by regular bores. The 4 axis Reverse Bore operation is invoked by selecting the Program tab, clicking on the Holes button in the Machining Browser and selecting the 4 Axis Rev Bore operation.

 

4 Axis Hole Making, Reverse Bore Menu Item

The Reverse Boring toolpath method is invoked by selecting the Program tab, clicking on the Holes button in the Machining Browser and selecting the Rev. Bore operation.

Reverse Bore Types

The reverse bore types supported are:

Reverse Bore  

The tool is fed to defined depth at the controlled feed rate; the spindle is stopped and then a rapid retract is performed.  It is then stopped to orient the spindle, moved away from the side of the hole and finally retracted.

User Defined Bore1 / User Defined Bore2

These are reverse bore cycles that you can define using the User Defined Cycles in the Post-Processor Generator.
