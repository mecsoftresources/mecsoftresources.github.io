Entry Motions

The Entry Motion section is used to control how the cutter approaches and enters during the cutting process.

None

Select None to not use an Entry Motion.  The tool will plunge into the stock at the current cut start point location.

No Entry Motion

Approach Motion / Vertical Dist (D)

Vertical Approach (D) allows you to specify a straight vertical engagement distance above the cut start point.  

Line/Arc

Select Line/Arc Entry to add a 3D Approach and Engage Motion to the entry of the toolpath.

Linear Entry

Approach Motion / Vertical Dist (D)

Vertical Approach (D) allows you to specify a straight vertical engagement distance above the cut start point.  

Linear

For Linear the cutter follows a linear ramp motion from the Vertical Distance (D) and engages the part at Angle (A).  Specify Length (L) and Angle (A).

Linear Entry

 

Length (L)

This is the Length (L) when engage is set to Linear.  It begins at the end of the approach motion and ends at the cut start point.  During this motion the tool travels at the Engage (Ef) feed rate.

Angle (A)

This is Angle (A) when engage is set to Linear.  It begins at the end of the approach motion and ends at the cut start point.  The angle is measured tangent to the cut start point.

Radial

For Radial, the cutter follows an arc with Radius (R) from the Vertical Distance (D) and engages the part at the cutting start point.  Specify Radius (R).

Engage Radial

Radius (R)

For Radial, the cutter follows an arc with Radius (R) from the Engage Ramp Height and engages tangent to the part.  Specify Engage Ramp Height and Radius (R).
Along Path

Define the entry as a ramp motion and then set the Along Path Angle (A) and the Along Path Height (H).  Refer to the image in the dialog and  below.

Along Path

Along Path Angle (A)

This is Angle (A) in the engage motion.  It begins at Height (H) and ends at the cut start point.  During this move the travels at the Engage (Ef) feed rate.

Along Path Height (H)

This is Height (H) in the entry motion.  It is the height at which the ramp entry motion begins.
Exit Motions

The Exit Motion section is used to control how the cutter retracts and departs during the cutting process.

None

Set Exit Motions to None to eliminate any exit motions.  The tool will retract from the stock at the end point on the toolpath.

Exit None

Departure Motion / Vertical Dist (D)

This linear motion, Vertical Distance (D) can be added to the end of the Retract Motion.  The cutter will perform the retract, then depart vertically for this distance at the Departure (Df) feed rate.

 
Line/Arc

Select Lines/Arcs to use a 3D exit for this operation.  Then specify the additional parameters that appear in this dialog.  

3D Exit Radial

Linear

For Linear the cutter follows a linear ramp motion from the endpoint of the cut at Angle (A) and for a distance of Length (L).  Specify Angle (A) and Length (L).

Retract Linear

Length (L)

Enter Length (L) for the linear retract motion that starts at the cut end point and ends at the departure start point.  During this motion the tool moves at the Retract (Rf) feed rate.

Angle (A)

Enter Angle (A) for the linear retract motion that starts at the cut end point and ends at the departure start point.  During this motion the tool moves at the Retract (Rf) feed rate.

Radial

For Radial, the cutter follows an arc with Radius (R) from the endpoint of the cut to the beginning of the Departure Motion.  Enter Radius (R).


Retract Motion: Radial

Radius (R)

Enter Radius (R) for the retract motion.  It begins tangent at the cut end point.  During this motion the tool moves at the Retract (Rf) feed rate.

Departure Motion / Vertical Dist (D)

This linear motion, Vertical Distance (D) can be added to the end of the Retract Motion.  The cutter will perform the retract, then depart vertically for this distance at the Departure (Df) feed rate.
