I lost my custom post from my previous product version!

Please follow these steps to retrieve your post processor from an older version in order to use it with your new product version

1.Use the Windows File Explorer to go to your Program Data folder.  This is typically called ProgramData and you can find it on your system drive - usually the C:\ drive. This folder is hidden, so you’ll only see it if you show hidden files set in your operating system.

2.Go to MecSoft Corporation/Older version product/Posts/MILL

3.Copy the post you were using by locating the postname.spm file in this folder

4.Now go to ProgramData/MecSoft Corporation/2021 product/POSTS/MILL/SPM/ folder

5.Paste the copied file here

Bring up your new product and you should be able to use your customized post-processor that you were using in your previous version.
