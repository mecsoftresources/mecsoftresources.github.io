Select Tools from Library

The Select Tools from Library button enables users to select tools from a previously saved Tool Library.

Locating the Load Tool Library button

 

The list of tools will now be displayed under Select Tools(s) from list dialog and you can drag and drop the tools from the selection list to the cutting tools browser.

 

Dialog Box: Select Tools(s) from list (Drag & Drop)

 

Dialog Box: Select Tools(s) from list (Drag & Drop)

 

To Edit, Rename, Cut, Copy or Paste on any of these tools, use right mouse button click after selecting the tool under Tools tab.
