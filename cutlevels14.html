Cut Levels tab, Curve Projection, 5 Axis

The following Cut Levels tab allow you to specify multiple passes which can be used as roughing operation.  It allows you to define the location of the Cut Depth Control and Cut Traversal parameters.  Refer to each of the sections below for more information.

 

Dialog Box: Cut Levels tab

Dialog Box: Cut Levels tab, 5 Axis Curve Projection Operations
Cut Depth Control

The Cut Depth Control enables you to define the depth of the cut.

Total Cut Depth  
Specify the Total Cut Depth by either typing in the desired value or by using the Pick icon and select 2 points on the part geometry. Use the object snaps to points on the geometry.

Depth/Cut  

The Depth/Cut can then be set by entering the desired value or by using the slider bar provided.
Cut Traversal between Cut Levels (if any)

This allows you to choose between Zig or Zigzag.

Zig  

Selecting Zig the cutter traverses the cuts in a single direction between levels.

ZigZag  

Selecting Zigzag the cutter traverses back and from along the cuts between cut levels.
