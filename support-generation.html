Support Generation

Use this command to generate a support structure for a selected mesh to properly support it during the 3D printing process.  An insufficiently supported mesh can collapse during the 3D print process due to overhangs and/or the weight of the print material.

Supports are generated based on an overhang value you provide and will extend from overhangs downward in the Z axis to the floor printer's bed.  Be sure to read this entire section carefully to fully understand the power of this command.

 

Read these Tips!

Support Generation

When to use this Command: ONLY use this command after Fit Mesh and Part Orientation as these commands confirm the final location and orientation of your part within the 3D printer's volume.

Good Practice:  It is ALWAYS a good practice to Diagnose and Auto Fix your mesh BEFORE using Support Generation. ANY errors such as gaps or self-intersects will have influence on the determination of overhang area and support generation.

Experimentation is Advised:  Review the Work Flow and Basic Procedure section below and then experiment with the various options and parameters.  You can adjust parameters and preview the support structure prior to committing them to a mesh.  The User's Guide of your 3D Printer may also provide recommended values for some of these parameters.

Understanding Support Structures:  This command will add support structures that will "connect" to the surfaces of your 3D Printed part.  Understand that these support structures will need to be removed manually once your 3D Print is completed.
Work Flow

1.First use the Fit and the Part Orientation commands to determine the best orientation for your part within the 3D printer's volume.  To determine the 3D printer's volume, first select a 3D printer from the list of supported printers and then toggle the display of the printer's volume On.

2.Select the Parameters tab of the dialog and set your desired parameters.

3.Select the Support Generation tab of the dialog to Generate the support preview.  You can generate the supports automatically and then add individual supports manually if needed.

4.Select the Details tab and review the information provided.

5.Once you are satisfied with the support structure, select the Create mesh from support structures tab, select an option and pick Create mesh(es).
Command Prompts

1.Select a mesh for which you want to create supports.

2.Select parameters and generate supports automatically or create them manually.

Support Generation Options

The following options and reference information are available from the Mesh Browser.

Parameters

The following parameters can be adjusted to achieve the desired support generation.  Once the values are set, go to the Support Generation tab (see below) and pick Generate.  The support structure will be previewed on your part.  You can refer to the images and definitions shown below for guidance.

Support Generation: Parameters tab

Support Generation Parameters

Overhang angle  
The overhang angle where support generation will occur (shown in blue on your part).

Minimum overhang height  
Any overhang lower than this height will be ignored in both the support calculation as well as the overhang calculation.

Max angle  
The maximum angle allowed between the Z axis and any support structure.  This determines the amount of overhang allowed for support structures.  

The Max angle should not exceed the Overhang angle (see above) or the generated support structure might be unprintable.

Support diameter  
The diameter of the support where it touches your part

Sampling distance  
The distance between support diameters on your part.  The smaller the value, the more supports that are generated.  The following images illustrate sampling distance.  The one of the left to too large while the one of the right is too large!

Sampling distance too large

Sampling distance too samll

Connector diameter  
The allowed diameter where the support connectors contact your part.

Connector height  
The Y distance between the Connector diameter (on your part) and any support structure.

Base diameter  
The diameter created at the base of each connector.

Base height  
The height of each base diameter.

Base connectors on mesh  
Select Allow, Disallow or Minimize
This option determines how many base connectors are allowed to contact your part.  In the image below on the left, all base connectors in the upper portion of the part are Allowed to connect to the part.  In the image on the right, base connectors are Disallowed to connect to the part.  The middle image Minimizes contact with the part.

A word about Base Connectors:  Because you will need to remove all support material manually, it might be desirable to Minimize or Disallow connectors contacting your part.  However, this may increase the amount of support material your printer will consume.  Also note that selecting the Disallow option may still produce connectors that contact your part in some specific cases when supports cannot be constructed without them.

Minimize connectors, Medium support material utilization, Medium part surface contact

Support Generation

The following support generation options are available:

Support Generation: Support Generation tab

Automatic Generation

Tree / Straight  
Select Tree or Straight to define the type of support structure to generate.  An example of each is shown below.

Tree

Straight

Add point to support  
Select this button and then pick a point on your part (within the blue overhang area) to add a sampling point.  Then pick Generate to display the new support structure.

Generate  
Pick this button to Generate a support structure based on the options selected.  It will be previewed your part.
Manual Generation

Add  
Select this button and then pick a point on your part (within the blue overhang area) to add a sampling point to the support structure preview.

Delete  
Select this button and then pick a connector or branch to delete it from the support structure preview.

Delete all generated structures  
Select this button to delete the current support structure.  Note: This can only be used while previewing the support structure before the Create Mesh(es) button is selected.  To delete the support structure after its mesh is created, select the mesh from the browser and use the Cut command from Rhino.

Create Mesh from Support Structures

The following options are available from the Create mesh from support structures tab:

Support Generation: Create mesh from support structures tab

Options

Create and unite with part
Select this option to create the support structure and unite its mesh objects with he part mesh object. Note: It is advised that you do not use this option unless you are absolutely sure your support structure is correct.

Create as separate mesh  
Select this option to create the support structure as a separate mesh object and list it in the Browser. Note: It is suggested that you use this option as it will be easier to select and delete the support structure mesh later if needed.

Create as separate mesh for each support tree
Select this option to create the support structure one separate mesh for each tree structure.  refer to the options and examples above for the difference between a Tree and a Straight support structure.  Even if Straight is selected, each straight support will be a separate mesh object.  All newly created mesh objects will be added to the Browser.

Create mesh(es)  
When you are satisfied with the support structure, select this button to commit it to actual mesh object(s). Note: Be sure you understand and select the desired Create option listed above as each has different results.
