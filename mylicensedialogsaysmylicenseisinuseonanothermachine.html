My License Dialog says my License is in use on another Machine!

If you are attempting to activate your MecSoft product license and the License Dialog says that my license is in use on another machine, here are some steps you can take to help resolve the issue:

•Obviously if you are sharing the license on multiple PCs (which is fine) please make sure all other PCs has the license released.  This can be done manually from the License dialog (Product Main Menu > License > Release product for use on another Machine.

•If you are certain that no other PC has your license active, then you can use our license clean up utility.  This utility cleans up temporary files related to licensing the product that can sometimes get corrupted if your PC experiences a crash and a recent system update.

 

To "Clean Up" License Temporary Files

In 2020:  
Type !!DT!! in the License Dialog and pick register.  It will cleanup any temporary files (located at: C:\ProgramData\MecSoft Corporation).  Then close and reopen the product and try to activate your license.

 
In 2019:
Permanently delete the various data files that are written there and then try activating again.
The data file names will all begin with "bc2f44...."  as shown below:
License Activation Failure (-2023)
