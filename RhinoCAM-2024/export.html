Export

When you're ready, use this command will save your mesh model to a compatible 3D printer file.  The File Save As dialog will display.  Select from one of 3 different 3D print file formats from the Save as type selector.  The file formats include Stereo-Lithography Binary (*.stl) files, Stereo-Lithography ASCII (*.sla) files and Additive Manufacturing (*.amf) files.

 

Command Prompts

1.Select mesh(es) to export to STL, SLA or AMF.:
Basic Procedure

 

Screen Pick

Optional Information

Step 1

Pick the Export icon from the 3D Print tab.

Steps 1 & 2 can be performed in reverse order.  You can select meshes first and then select the command icon.

Step 2

Select the mesh you want to export.

Step 3

From the File Save As dialog, select the directory and file type to save to.  The default directory and file name will be the same as the current file.

3D Print Support for Windows® 8.1 (or higher)

RhinoCAM-MESH supports Windows® 8.1 (and higher) printer drivers.   If you are running Windows® 8.1 with a 3D Printer and driver installed on your machine, refer to the Print command to print directly to your attached 3D printer.
