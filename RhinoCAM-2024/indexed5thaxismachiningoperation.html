Indexed 5 Axis Machining Operations

Shown below is a typical part that would require index 5 axis milling operations to manufacture the part.  Notice that the part has undercut geometry when looking down from the top. This means that there are areas of the part, which cannot be reached by a straight tool moving purely up and down.  To successfully machine this part, it will use 2 or more different orientations by using Setup and Set Work Zero operations to orient the tool normal to a planar surface.

Indexed 5th Axis Machining Operations

It should be noted that MILL module allows for any user defined indexing angle and is not restricted to 180 degrees.  This feature allows you to machine multi-sided parts.

Specifying rotation axis and rotation center

Before creating 5 axis toolpaths, you need to specify to the system the primary and secondary rotation axis. This can be done by selecting the Machine Tool button in the Setup window of the Browser.  Refer to Machine Setup for detailed description.
