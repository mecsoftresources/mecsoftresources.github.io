Cycle G-Code

Optimize Cycle Options

Optimize Cycle output will define the cycle format only once and will output the X,Y values for all the other holes. This will result in significant reduction in the file size for output.

•Optimize Cycle Output:  Enables the optimized cycle options.

•Cycle Coordinate Block:  Specific cycle block variables to use.
Dwell Options

•Scale Factor for Dwell:  Sets the scale factor for dwell output.

•# of Decimal Places:  Sets the number of decimal places for output.
Thread First Depth Options

•Scale Factor:  Sets the scale factor for the first thread depth.

•# of Decimal Places:  Sets the number of decimal places for output.
