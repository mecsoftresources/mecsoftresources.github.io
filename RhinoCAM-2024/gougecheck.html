Gouge Check, Milling Operations, 5 Axis

During 5 Axis operations, the following Gouge Check tab is used to avoid tool and holder collisions with the part being machined. You specify a Gouge Check Strategy, Tool Geometry and Check Geometry parameters.  Refer to each of the sections below for more information.

 

Dialog Box: Gouge Check tab

Dialog Box: Gouge Check tab, 5 Axis Milling Operations
Perform Gouge Check

Check this box to enable the Gouge Checking against Check Geometry for the current 5 Axis operation.  

You will then be able to:

•Set tool geometry to be checked for gouge

•Set geometry to be checked for gouge

•Set strategy for gouge check
Tool Geometry

Tool Geometry can be checked against the surface geometry for Tip, Shaft and Holder. When Shaft and Holder geometry options are selected, you can then specify the Tool Shaft Clearance and Holder Clearance parameters.
Check Geometry

Check Geometry allows you to select the types of geometries for collision avoidance during toolpath computation. Selecting Drive Surfaces, only checks for collisions with the surfaces selected for machining. Selecting Check Surfaces, allows you to select surfaces to avoid.
Check Surfaces

Selecting Check Surfaces allows you Add and Remove surfaces to check for collision avoidance.

Add

Selecting Add button minimizes the dialog and prompts you to select surfaces. To end the selection press the enter key on the keyboard or right mouse button click.  The selected surfaces are displayed under Check Surfaces list. Selecting a surface from the list highlights the geometry on the graphics area and displays the surface normal.

Check Geometry, dialog

Check Geometry, highlighted

Remove All

Removes all Selected Surfaces from the Check Surfaces list.

Remove Active

This allows you to remove surface from the selection list. This can be done by selecting a surface from the list of Check Surfaces and click Remove Active or press the delete key on the keyboard.
Gouge Check Strategy

This section of the Gouge Check tab allows you to apply a method to the toolpath when a collision or gouge is detected.  The following strategies are the available:

Retracting tool along tool axis

In this strategy the tool retracts along tool axis as it approaches the check surfaces.

Retracting tool along tool axis
Moving Tool away

In this strategy the tool moves away by retracting the tool in one of specified orientations.

Tilting tool away with max angle

In this strategy the tool tilts by a given angle to avoid check surfaces.  This can be set by selecting one of the following methods.

Use Lead/Lag angle  

Use side tilt angle  

Use lead/lag and side tilt angle  

Selecting one of the above methods allows you to specify a positive, negative or both positive and negative angle value.
