Cutting Tools

The MILL module allows you to define various types of milling and drilling tools.  The tool types that are currently supported are listed in the table below. Vertical and Tapered Tool Holders are also supported.  can be define Ball, Flat, Corner-radius or bull, VeeMill, Chamfer, Taper, Thread, Face, Dove Tail, Fillet, Lollipop, User Defined, Drill tools, Center drills, Reamer, Tap, Bore and Reverse Bore Mills.  The tool and holder will participate in collision detection and other functions involving the tools.
