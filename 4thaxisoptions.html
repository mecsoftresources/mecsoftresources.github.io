4 axis Machining Operations Options allows you to set 4 axis options.

 

 

4 Axis Options Menu Item

The 4 Axis Options are invoked by selecting the Program tab, clicking on the 4 Axis button in the Machining Browser and selecting Options.

4 Axis Options Menu Item
Dialog Box: 4 Axis Machining Operations Options

This brings up the 4 Axis machining operations options dialog.

Dialog Box: 4 Axis Machining Operations Options

 

Always retract tool to Tool Change Point before each table rotate operation

When you select this option, MILL module will retract the tool to the tool change point. The tool change point is specified under Machine Tool Setup located under the Program tab in the Machining Browser.

Retract to Tool change point before each table rotate is output in the posted code only when a non-zero value for X Y Z coordinates is specified under Tool change point in the Machine Setup dialog.
 
Refer to Machine Setup
Always warn if rotation center is not aligned with X or Y axis

Check this box to be warned if the Machine Zero and the rotary center are not set at the same coordinate point.  You can override this warning by un-checking this option.
Do not output intermediate points for transfer motions

When this option is checked, and Cut Transfer is set to Rapid, intermediate points will not be output during transfer motions in 4 Axis continuous toolpaths.  This allows the machine tool to rotate the part using an arc motion rather than using the linearized transfer motions that are computed by the CAM system.
