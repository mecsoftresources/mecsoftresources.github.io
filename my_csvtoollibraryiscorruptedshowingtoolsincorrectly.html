My *.csv Tool Library is Corrupted Showing Tools Incorrectly

When you save a new Tool Library it is preferred that you use the *.vkb format (do not use the *.csv format).  Here are some issues that can arise from using the *.csv format for saved Tool Libraries and how to fix them:

1.Do not save a Tool Library as a *.csv formatted file.

2.The *.csv Tool Library format DOES NOT save the feeds and speeds for each tool!

3.If your *.csv Tool Library file when loaded, shows tools incorrectly.  Tools are missing and some tools are defined as a different tool!  Go to your Windows file associations and associate all *.csv files with the program Notepad.exe.
 
Here is how to do it:
a) Select the csv file from the File Browser in Windows.
b) Right-click on the csv file and select Properties.
c) From the General tab, find the "Opens with" field and select the Change button.
d) from the pop up dialog to select a program, go to Other Programs and select the program called Notepad and then pick OK to close the dialog.
e) Pick OK to close the File Properties dialog also.
 

4.Now load the *.csv library in your MecSoft CAM plugin and check all tools for correctness and add feeds and speeds if needed, then save the Tool Library out as a *.vkb file.
