Entry / Exit

The Entry/Exit section can be used to control how the cutter enters and leaves during the cutting process.

•Approach Motion / Vertical Distance (D)  
In this option the approach and departure Vertical Distance (D) can be specified. This is the distance from the cut start point where the cutter switches from an approach feed rate to an engage feed rate for entry and from retract feedrate to departure feedrate for exit motions.

•Engage Motion  
The Approach and Retract motions can be Linear or Radial. The cutter uses a ramp motion with the defined Angle (A) and distance Length (L) away from the cut start point to engage and retract from the cut start or end point. The ramp angle is measured from the horizontal.
In the Radial engage/retract option, a circular arc motion is used to engage and retract from the cut start or end point. The Radius (R) of this arc can be controlled.

Cut Connections

Cut Connections can be turned on for smooth transition between cuts (stepover). The transfer between cuts takes place at the end of each cut.  Cut connections can be Straight, Linear or Radial.  You can specify the Length (L), Angle (A) or Radius (R) for the cut connections.

Linear

Radial

The Add Loops option can be enabled for Linear or Radial cut connections. These transitions allow the cutter to transition from one cut to the next in a smooth manner thereby reducing rapid acceleration and deceleration on the machines.

Radial + Add Loops

The option to specify Cut Connections are only available in some of the 3 axis machining operation types.
