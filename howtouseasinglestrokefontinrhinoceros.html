How to: Single Stroke Font

Click here to Download the single stroke font and follow these instructions:

1.Download and unzip the attached font file to a temporary folder on your computer.

2.Copy MecSoft_Font-1.ttf to C:\Windows\Fonts. The font will now be installed on your computer.

3.Open Rhino.

4.In the Rhino command prompt, type the TextObject command.

Rhino TextObject command dialog

5.Make sure to select Curves.

6.Select MecSoft_Font-1 from the list of fonts.

7.Pick OK to create the Rhino curve objects.
